How to read this

  • Orange means the current minimum candidate for this pass.
  • Yellow scans through the remaining unsorted values and checks if a new minimum is found.
  • At the end of each pass, the minimum is swapped to the front and turns green (sorted).
  • Selection Sort still does all comparisons in every case, so its time complexity stays O(n^2).
